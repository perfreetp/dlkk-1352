# AI 互聊 - 产品需求文档

## 1. 产品概述

AI 互聊是一款面向写作者和社群运营者的纯前端工具，帮助用户快速创建多个 AI 角色并模拟多角色对话场景。用户可以自定义角色性格、关系，创建群聊或一对一对话，生成剧情，收藏精彩片段，导出对话稿，为创作和社群运营提供灵感和素材。

- 目标用户：小说作者、剧本创作者、社群运营者、内容创作者
- 核心价值：快速模拟多角色互动，激发创作灵感，提升内容创作效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 创建/编辑/删除 AI 角色，创建聊天室，使用全部功能 |

### 2.2 功能模块

1. **角色广场**：浏览和管理所有 AI 角色，支持创建、编辑、删除角色
2. **我的 AI**：管理个人 AI 角色库，支持模板化管理
3. **聊天室**：多 AI 自动对话，支持旁白插入、暂停、改写、固定记忆等
4. **关系网**：可视化展示角色之间的亲密度与立场关系
5. **剧本板**：按主题生成开场，管理对话剧本
6. **回放库**：收藏精彩对话记录，查看发言占比统计
7. **设置**：应用设置和模板管理

## 3. 核心流程

### 3.1 创建 AI 角色流程
用户进入角色广场 → 点击创建角色 → 填写角色信息（姓名、语气、记忆、禁忌、头像）→ 保存角色 → 角色出现在角色广场

### 3.2 开始聊天流程
用户进入聊天室 → 创建房间 → 选择多个 AI 角色加入 → 设置聊天模式（群聊/一对一）→ 生成开场或手动开始 → AI 自动轮流发言 → 用户可随时暂停、插入旁白、改写某句

### 3.3 收藏与导出流程
聊天过程中收藏精彩片段 → 进入回放库查看 → 可导出为文本文件

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深靛蓝 (#1e1b4b) 为主色调，搭配暖金色 (#fbbf24) 作为强调色，营造创作氛围
- **辅助色**：柔和的紫色渐变背景，营造神秘而富有创意的视觉体验
- **按钮风格**：圆角胶囊形按钮，带有微妙的悬浮动效
- **字体**：标题使用「思源宋体 / Noto Serif SC，体现文学感；正文使用「Inter / 系统无衬线字体，保证可读性
- **布局风格**：侧边栏导航 + 主内容区，卡片式设计，毛玻璃效果
- **图标风格**：线性图标，简洁现代
- **整体氛围**：暗色调为主，营造沉浸式创作体验

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 角色广场 | 角色卡片 | 头像、姓名、标签、语气标签 |
| 角色广场 | 创建表单 | 输入框、文本域、头像选择器、标签输入 |
| 聊天室 | 对话区域 | 聊天气泡、头像、姓名、时间 |
| 聊天室 | 控制面板 | 播放/暂停按钮、旁白按钮、设置按钮 |
| 聊天室 | 侧边栏 | 在线角色列表、发言统计 |
| 关系网 | 图谱区域 | 节点（角色头像）、连线（关系线）、力导向布局 |
| 剧本板 | 剧本列表 | 卡片式布局，主题标签 |
| 回放库 | 记录列表 | 时间、标题、参与角色、收藏按钮 |
| 设置 | 设置项 | 开关、滑块、选择器 |

### 4.3 响应式设计

- 桌面端优先设计，侧边栏导航
- 平板端：侧边栏可收起
- 移动端：底部导航栏，内容区自适应

### 4.4 动效设计

- 页面切换：淡入淡出 + 轻微滑动
- 聊天气泡：入场动画
- 角色卡片：悬浮放大效果
- 关系网节点：平滑的力导向动画
- 按钮交互：按压反馈动效
